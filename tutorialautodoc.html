Módulo AutoDoc

🚀 Tutorial: CMS Inteligente

"Se você sabe preencher uma tabela, você sabe criar mil documentos."

Este guia explica como transformar dados de uma planilha em documentos profissionais (PDF ou Word) de forma automática, personalizada e organizada.

01

O Conceito: A "Mágica" das Etiquetas (Tags)

O segredo de tudo está nas Tags. Imagine que seu documento é um formulário com espaços em branco. Em vez de deixar o espaço vazio, você coloca uma "etiqueta" entre chaves: {{Nome}}.

A Planilha

É onde guardamos as informações (O combustível).

O Template

É o seu modelo de documento no Google Docs (O molde).

O AutoDoc

É o motor que lê a planilha e carimba as informações no lugar certo das etiquetas.

02

O Esqueleto do Sistema (Como funciona a lógica)

Para usar o sistema, você só precisa de três ingredientes:

  • Uma Planilha com Títulos: A primeira linha da sua planilha deve ter nomes claros (ex: Nome, Data, Valor, Cargo).
  • Um Modelo (Template): Um arquivo no Google Docs formatado do seu jeito. Onde você quer que a informação da planilha apareça, escreva o título da coluna entre chaves. Ex: Eu, {{Nome}}, portador do CPF {{CPF}}...
  • Uma Pasta de Destino: Onde o sistema vai "cuspir" os arquivos prontos.
💡 Regra de Ouro: O que está dentro do {{...}} no documento deve ser exatamente igual ao título da coluna na planilha. Se na planilha está "Nome Completo", no documento deve estar {{Nome Completo}}.
03

Passo a Passo: Criando sua Automação

Passo 1: Configuração (Onde e Quem)

Nome da Automação: Dê um nome para o seu "robô" (ex: "Gerador de Ofícios 2026").

Busca de Arquivos: Use a lupa 🔍 para encontrar seu modelo e a pasta onde os arquivos serão salvos. O sistema faz a busca no seu Google Drive para você não ter que copiar links complicados.

Passo 2: Verificação de Tags (O Check-up)

O AutoDoc vai ler seu documento e avisar: "Olha, encontrei a tag {{Data}}, e ela existe na sua planilha. Tudo certo!".

Se aparecer algo em vermelho, significa que você esqueceu de criar aquela coluna na planilha ou escreveu o nome diferente.

Passo 3: Formatos e Envio (O Toque Final)

Nome do Arquivo: Você pode personalizar o nome de cada arquivo gerado usando as tags! Ex: Contrato - {{Nome}}. Isso evita aquela pasta cheia de arquivos com o mesmo nome.

PDF ou Doc: Escolha se quer o arquivo editável ou fechado em PDF.

E-mail Automático: Se você marcar esta opção, o sistema envia o documento pronto para o e-mail que estiver na planilha.

04

Cuidados e Boas Práticas (Para não errar)

⚠️ Status Doc: O script cria uma coluna chamada "Status Doc". Se estiver escrito "Sucesso" lá, o robô não vai processar aquela linha de novo. Isso evita enviar o mesmo documento duas vezes para a mesma pessoa. Se quiser refazer, apague o "Sucesso".
⚠️ Letras Maiúsculas/Minúsculas: O sistema é sensível. {{nome}} é diferente de {{Nome}}. Mantenha o padrão.
⚠️ Limpeza de Dados: Use as ferramentas de "Formatação de Texto" no menu 🧠 CMS Inteligente para garantir que os nomes fiquem bonitos (ex: converter tudo para "Nome Próprio" antes de gerar os documentos).
05

O Poder da Abstração: "O céu é o limite"

Não pense que o AutoDoc serve apenas para contratos. Se você entender a lógica das Tags, pode criar:

Certificados de Cursos:
(Tags: {{Aluno}}, {{Carga Horária}})
Editais de Convocação:
(Tags: {{Conselheiro}}, {{Pauta}}, {{Hora}})
Recibos:
(Tags: {{Valor}}, {{Referente a}})
Relatórios Mensais:
(Tags: {{Mês}}, {{Indicador 01}})

O sistema é uma ferramenta vazia: a inteligência e a utilidade vêm da sua necessidade.
